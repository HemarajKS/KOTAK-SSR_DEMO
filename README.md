# Expo SSR App

React Native app with server-side rendering using Expo.

## Setup

```bash
npm install
```

## Development

```bash
npm run web
```

## Build & Deploy

```bash
npm run build
npm run serve
```

## Features

- Server-side rendering
- Client-side hydration
- React Native components on web
- Cross-platform compatibility# KOTAK-SSR_DEMO
